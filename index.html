Imago - Social Media Webseite

Imago ist eine Social Media Webseite, wo User ein Profil erstellen können. Auf dieses können sie Text posten oder Bilder hochladen, die von anderen Usern geliked werden können. Der User kann auch Freunde hinzufügen, um dessen Profile schneller abrufen zu können.

Bookstore App - Mobile App

Bookstore ist eine, wie der Name schon sagt, eine Büchershop App (oder eher eine Simulation davon). Sie wurde mit Kotlin in Android Studio geschrieben. Die Bücher, die in der App erhältlich sind, sind Coding Bücher und werden aus einer API gefetched. Auf den Screenshots sind man die Wireframes der App.
